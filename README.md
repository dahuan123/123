# 读书
    
 我猜，作为一个普通人，你肯定不止一次听过一个令人心惊胆颤的词汇：阶层固化。<br><br><br><br>
 
 
 
 
 
无论阶层固化是否真的已经存在，无论它是否真的正在逐步形成，它都肯定是令普通人一听就心头一紧的词汇。我不知道别人是怎样的感觉，反正我清楚地记得年轻的自己第一次听到这个词汇的时候那种接下来很久都忧心忡忡的感觉 —— 那不是恐惧，那好像是绝望。<br>
